# Three Layers of Rigor

## Layer 1: Contract Rigor

Every task must state:

- preconditions
- postconditions
- invariants
- explicit assumptions

## Layer 2: Dynamic/System Rigor

Stateful, concurrent, or temporally constrained tasks require:

- state variables
- transition relation
- safety properties
- liveness properties
- executable model checking

## Layer 3: Structural/Compositional Rigor

When the core challenge is composition, abstraction, or universal structure, add:

- categorical interpretations
- refinement mappings
- compositional laws

Layer 3 must strengthen the pipeline, not merely decorate it.
