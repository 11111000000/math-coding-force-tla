Artifact-Centered

Чат — transient. Пакет — источник истины. Нет файла — нет существования.

Никаких «экспертных оценок». TLC проверяет каждое состояние.

没有"专家估计"。TLC详尽检查每个可达状态。

From TLA+ to TypeScript От TLA+ к TypeScript 从TLA+到TypeScript

The same state machine, formalized and implemented. Один и тот же автомат, формализованный и реализованный. 同一个状态机，经过形式化和实现。

Model.tla
VARIABLES state, pendingResult

Init ==
  /\ state = "closed"
  /\ pendingResult = "none"

Open ==
  /\ state = "closed"
  /\ state' = "opening"
  /\ UNCHANGED pendingResult

Confirm ==
  /\ state = "open"
  /\ state' = "confirming"
  /\ pendingResult' = "in-flight"
dialogReducer.ts
export function dialogReducer(
  state: DialogStateShape,
  action: DialogAction,
): DialogStateShape {
  switch (action.type) {
    case "OPEN":
      if (state.state !== "closed") return state;
      return {
        state: "opening",
        pendingResult: { kind: "none" }
      };
    // ... 10 actions total
  }
}
